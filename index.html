    How this estimate works

    The probability uses historical data: we compute the average time between consecutive sirens at each place, then see how the current "time since last siren" compares. Under the average gap → higher probability; over → lower. We also add time-of-day progression, distance from the last siren to your location, and a clustering bonus for multiple recent alerts. Result is smoothed and clamped between 0.5% and 99.4%.

    Data from Pikud HaOref. With the local server (run node server.js in local-pusher/), probability is computed on your PC and the chart shows history. Otherwise the dashboard uses GitHub-hosted data or demo data. Not an official alert system.
